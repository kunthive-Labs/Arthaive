India's funding data, finally open.

A searchable database of Indian startup funding rounds where every record links back to its original news source.

Open the platform View on GitHub

Funding intelligence in India is either expensive or unverifiable.

Understanding Indian startup funding today means paying thousands of dollars a year for Tracxn, settling for Crunchbase's shallow India coverage, or hand-scraping news sites into spreadsheets and losing track of where each number came from.

Arthaive is the free, India-first alternative. Every record in the database links back to the news article it was extracted from. If a number is wrong, you can click through and check. The platform stays free, and the whole project is open: code under AGPL-3.0, data under CC BY-SA 4.0.

Built for exploring the data, not just listing it.

explore

Every deal, filterable and traceable

Filter funding rounds by sector, stage, city, amount, and date. Every row links to the article it was extracted from.

analytics

The Indian funding landscape on one dashboard

Funding trends over time, sector breakdowns, stage funnels, investor leaderboards, and a map of activity across India.

profiles

Investors and sectors in depth

Each investor and sector gets a profile with its full deal history and the network of co-investors it appears alongside.

search

Ask in plain language

Natural-language search over the dataset, with a deterministic fallback that keeps queries working when AI is not configured.

public api

Build on the data

A public REST API v1 with versioned endpoints and free keys, so researchers and developers can query the same records the platform uses.

source-backed

No sourceless records, by design

A record without a source never enters the database, and coverage gaps are disclosed honestly instead of papered over.

Running in two minutes.

Clone, install, and browse. The public data pages need no environment variables, and the same records are one curl away through the API.

arthaive - zsh
$ git clone https://github.com/kunthive-Labs/Arthaive.git $ cd Arthaive && npm install $ npm run dev # public data pages need no env vars # query the public api v1 $ curl 'https://arthaive.kunthive.in/api/v1/funding-rounds?sector=Fintech&stage=series_a&limit=5' \ -H 'X-API-Key: ifk_your_key'

Open on purpose.

Funding data only earns trust when anyone can audit it. Arthaive keeps the record, the pipeline, and the licenses open so that trust never has to be taken on faith.

Every number is checkable

Each record carries a link to the news article it came from. Verification is a click, not a support ticket.

Free stays free

The platform, the data, and the API keys cost nothing. There is no paid tier waiting behind the search box.

Dual-licensed openness

The code is AGPL-3.0 and the data is CC BY-SA 4.0, so improvements and derived datasets stay open too.
